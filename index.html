Few-Data Speaker Adaptation Text-to-Speech using Style Equalizer

Yongmin Kim, Kyungseok Oh, Jeongki Min, Junyeop Lee, Bonhwa Ku, and Hanseok Ko



ABSTRACT

Adaptive text-to-speech (ATTS) aims to adapt a source TTS model to synthesize a new target speaker's speech using few speech data based on finetuning.
